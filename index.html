Setting Up an Anonymization

Map each table's column you want to anonymize with an Anonymizer. 


						console db-tools:anonymization:list

						-------------------- --------------------------------------------------------------------------------------------------------------------------------
							core
						-------------------- --------------------------------------------------------------------------------------------------------------------------------
							address              Anonymize a mutlicolumn postal address.
																	 Map columns for each part with options ('country', 'locality', 'region', 'postal_code', 'street_address', 'secondary_address')
							email                Anonymize email addresses. You can choose a domain and a tld with option 'domain'.
							firstname            Anonymize with a random firstname from a sample of ~1000 items.
							float                Anonymize with a random float between two bounds.
																	 Options are 'min' , 'max' and 'precision' (default 2).
							iban-bic             Anonymize IBAN and BIC on two columns.
																	 Map columns for each part with options ('iban' and 'bic')
							integer              Anonymize with a random integer between two bounds.
																	 Options are 'min' , 'max'.
							lastname             Anonymize with a random lastname from a sample of ~1000 items.
							lorem                Replace a text with lorem ipsum.
																	 Available options:
																		- 'paragraphs': (int) number of paragraphs, default is 1,
																		- 'html': (bool) surround each paragraph with \, default is false.
																		- 'sample_count': (int) how many different values to use (default is 100).
							md5                  Anonymize a column by hashing its value.
							password             Anonymize a password erasing it with a given value.
																	 Options are 'algorithm' (default 'auto') and 'password' (default 'password').
							string               Anonymize a column by setting a random value from a given 'sample' option.

Setting Up an Anonymization

On Doctrine Entities, from a list of various anonymizers


					namespace App\Entity;

					use Doctrine\ORM\Mapping as ORM;
					use MakinaCorpus\DbToolsBundle\Attribute\Anonymize;

					#[ORM\Entity()]
					#[ORM\Table(name: 'customer')]
					class Customer
					{
							#[ORM\Id]
							#[ORM\GeneratedValue]
							#[ORM\Column]
							private ?int $id = null;

							#[ORM\Column(length: 180, unique: true)]
							#[Anonymize(type: 'email')]
							private ?string $emailAddress = null;

							#[ORM\Column]
							#[Anonymize(type: 'integer', options: ['min' => 10, 'max' => 99])]
							private ?int $age = null;

							#[ORM\Column(length: 255)]
							#[Anonymize(type: 'string', options: ['sample' => ['none', 'bad', 'good', 'expert']])]
							private ?string $level = null;

							#[ORM\Column(length: 255)]
							#[Anonymize(type: 'md5')]
							private ?string $secret = null;

							#[ORM\Column]
							private ?\DateTime $lastLogin = null;

							// ...
					}
